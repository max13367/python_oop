# ЛР-7 — Консольное приложение банковской системы

## Цель работы

Разработать полнофункциональное консольное приложение (CLI) для управления банковскими счётами, объединяющее знания из ЛР1–ЛР6:

- Иерархия классов (`ClientAccount`, `SavingsAccount`, `CreditAccount`)
- Коллекции и операции поиска / фильтрации / сортировки
- Обработка исключений
- Работа с файлами (JSON)
- Типизация и документирование кода

## Структура проекта

```
lab07/
├── main.py          # Точка входа, запуск CLI
├── cli.py           # Интерфейс: меню, ввод/вывод (без бизнес-логики)
├── app.py           # Все операции над коллекцией счётов
├── storage.py       # Сохранение/загрузка в JSON
├── exceptions.py    # Собственные исключения
├── accounts.json    # Хранилище данных (создаётся автоматически)
└── README.md
```

| Файл | Назначение |
|------|------------|
| `main.py` | Точка входа; вызывает `cli.run()` |
| `cli.py` | Только ввод/вывод; никакой бизнес-логики напрямую |
| `app.py` | Все операции над коллекцией; CLI обращается только сюда |
| `exceptions.py` | Собственные исключения предметной области |
| `storage.py` | Сохранение/загрузка через JSON |

## Описание CLI

### Главное меню

```
==================================================
        БАНКОВСКАЯ СИСТЕМА
==================================================
1.  Показать все счета
2.  Добавить счёт
3.  Найти счёт по ID
4.  Найти по владельцу
5.  Удалить счёт
6.  Пополнить счёт
7.  Снять средства
8.  Заблокировать счёт
9.  Фильтровать по валюте
10. Фильтровать по балансу
11. Фильтровать по типу
12. Сортировать счета
0.  Выход
==================================================
```

### Функциональность пунктов меню

| № | Действие | Описание |
|---|----------|----------|
| 1 | Показать все счета | Вывод всех счётов с форматированием |
| 2 | Добавить счёт | Диалог выбора типа и ввода данных |
| 3 | Найти счёт по ID | Поиск по уникальному идентификатору |
| 4 | Найти по владельцу | Регистронезависимый поиск по имени |
| 5 | Удалить счёт | Удаление с подтверждением |
| 6 | Пополнить счёт | Добавление средств |
| 7 | Снять средства | Снятие с проверкой баланса |
| 8 | Заблокировать счёт | Блокировка операций |
| 9 | Фильтр по валюте | Вывод счётов в указанной валюте |
| 10 | Фильтр по балансу | Вывод счётов в диапазоне баланса |
| 11 | Фильтр по типу | Вывод счётов определённого типа |
| 12 | Сортировка | Выбор стратегии сортировки |
| 0 | Выход | Сохранение данных и завершение |

### Обработка ошибок

**Собственные исключения:**

| Исключение | Описание |
|------------|----------|
| `AccountNotFoundError` | Счёт не найден |
| `DuplicateAccountError` | Счёт с таким ID уже существует |
| `AccountBlockedError` | Счёт заблокирован |
| `InsufficientFundsError` | Недостаточно средств |

**Обработка ввода:**

- `ValueError` — при вводе строки вместо числа
- Все ошибки выводятся и не прерывают работу программы

Примеры сообщений:

```
Ошибка: введите число!
Счёт с ID '1000000001' уже существует
Счёт с ID '9999999999' не найден
```

### Сохранение и загрузка данных

**При запуске** — автоматическая загрузка из `accounts.json`:
- Если файл не существует — создаётся пустая коллекция
- Если существует — восстанавливаются все счёта с их типами

**При выходе** — запрос подтверждения и сохранение:

```
Сохранить данные перед выходом? (y/n): y
Данные сохранены.
До свидания!
```

**Формат JSON:**

```json

    "currency": "RUB",
